Это сервер для теплицы

1. В этом проекте есть папка src/

2. В этой папке есть главный файл main.cpp (при запуске в Arduino IDE его нужно переименовать в main.ino)

3. Также в этой папке src/ есть 2 папки src/scripts и src/server

3.1 В папке /scripts лежат все основные заголовочные файлы для каждой стрницы

3.2 В папке /server лежат html файлы для разметки
3.2.1 При редактировании этих html файлов важно учитывать несколько моментов:

1 - Всегда нужно использовать одинарные кавычки, т.е. ' вместо "

2 - В одном файле должны прописываться и CSS стили, HTML разметка и JS функции (в том же порядке, в котором я их перечислил)

Важно!
После того как вы отредактируете какой нибудь html файл, например main.html, вы должны перейти на сайт https://letteros.com/shorthtml/
и там вы должны полностью вставить свой отредактированный HTML файл, нажать СЖАТЬ, скопировать появившийся результат справа.
После того как вы скопировали сжатый о 1 строки файл, вы должны открыть заголовочный файл, в котором лежит разметка, которую вы хотели
изменить, и вставляете в переменную String output = "ваш сжатый до 1 строки код".

Для чего это нужно. Со временем файлы разметки будут увеличиваться с добавлением новых функций, стилей или элементов разметки.
И прописывать все это в Arduino IDE не эффективно, поэтому всю разметку вы делаете в редакторе кода (например VSCode).
Этот метод очень ускоряет работу, т.к. во всех редакторах кода есть функция подсказок или автозаполнений, но этих функция нет в Arduino IDE,
и это на самом деле очень замедляет работу в этой среде разработки, поэтому я придумал этот метод разработки, который ускорил мою работу
в разы. Поэтому я прошу вас тоже придерживаться этого метода.

Благодарю за внимание!
